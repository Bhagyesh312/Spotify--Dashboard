# 🎧 Spotify Music Dashboard (Power BI)

## 📊 Overview

This project is an interactive Spotify Music Dashboard built using Power BI.
It provides insights into songs, artists, genres, and audio features.

---

## 🚀 Features

* 🎵 Total Songs, Artists, Genres KPIs
* 📈 Popularity Trend Analysis
* 🎤 Top Artists & Genre Distribution
* 🎧 Audio Features Comparison
* 🖼️ Artist Image Slicer (Interactive)
* 🔄 Reset Filters Button

---

## 🛠️ Tools Used

* Power BI
* Excel / CSV
* DAX

---

## 📁 Dataset

Dataset includes:

* Songs metadata
* Artist information
* Audio features (energy, danceability, etc.)

---

## 🎯 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Ensure dataset paths are correct
4. Interact with filters and visuals
